AWS Static Portfolio Website

Project Overview

This project demonstrates hosting a responsive static portfolio website using Amazon S3 and Amazon CloudFront with HTTPS enabled.

Technologies Used

* HTML5
* CSS3
* Google Fonts
* Amazon S3
* Amazon CloudFront
* Git & GitHub

## AWS Services Used

Amazon S3

* Created an S3 bucket for website hosting
* Uploaded website files
* Configured static website hosting

Amazon CloudFront

* Created a CloudFront distribution
* Connected CloudFront to the S3 bucket
* Enabled HTTPS delivery


Deployment Steps

1. Created the portfolio website using HTML and CSS.
2. Created an S3 bucket.
3. Enabled Static Website Hosting.
4. Uploaded website files to S3.
5. Created a CloudFront distribution.
6. Set index.html as the default root object.
7. Verified website accessibility through CloudFront.
